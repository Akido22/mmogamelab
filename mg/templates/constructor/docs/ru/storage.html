Статическое хранилище

Чтобы иметь возможность загружать произвольные файлы на сервер, администратор игры может подключить соответствующий модуль через интерфейс "Системные модули игры". Эта функция может быть полезной, чтобы выложить какие-то дополнительные материалы об игре. Например, документы в PDF, таблицы в Excel, саундтрек в mp3, дополнительные картинки, на которые можно ссылаться через HTML-теги или любые другие данные.

Чтобы загружать объекты объёмом более 30 килобайт, владелец игры должен иметь Начальный аттестат WebMoney (или выше). Это необходимо, чтобы конструктор игр не использовали как склад пиратского контента.

Максимальный размер одного объекта — 3 мегабайта. Если вам необходимо загружать файлы большего объёма, хранилище должно оплачиваться. Для заказа этой возможности воспользуйтесь Аукционом заявок. Исключение составляют Flash-ролики (файлы .swf) и mp3-файлы (.mp3), для которых установлен лимит 20 мегабайт.
