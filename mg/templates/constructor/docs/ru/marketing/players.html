Статистика по пользовательской базе онлайн-игры

Административный интерфейс со статистикой по базе игроков вашей игры доступен в меню "Маркетинг / Статистика по пользовательской базе":

  • Период — дата, за которую представлена статистика. Статистика снимается на момент окончания календарных суток по Гринвичу;
  • CCU (Concurrent Users) — пиковое значение "онлайна" за сутки;
  • Нов — количество пользователей, впервые посетивших титульную страницу игры (входящий трафик);
  • Зарег — количество новых зарегистрированных игроков;
  • Ушло — количество ушедших (не заходивших в игру более 14 суток) игроков;
  • Возвр — количество вернувшихся (после ухода) игроков;
  • Акт — количество активных (заходивших хотя бы раз за последние 14 суток) игроков на момент конца суток;
  • DAU (Daily Average Users) — количество уникальных игроков, заходивших в игру в течение последних суток;
  • WAU (Weekly Average Users) — количество уникальных игроков, заходивших в игру в течение последних 7 суток;
  • MAU (Monthly Average Users) — количество уникальных игроков, заходивших в игру в течение последних 30 суток.
