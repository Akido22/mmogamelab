Страница с выбором класса персонажа генерируется из двух файлов: charclass-select.html и charclass-select-layout.html. В них можно использовать следующие переменные шаблонизатора:

  • classes — список классов. Каждый элемент списка — запись с полями:
    • id — идентификатор класса;
    • name — название;
    • description — описание;
  • error — текст сообщения об ошибке (может отсутствовать);
  • layout — присутствует только в шаблоне charclass-select.html. В это место подставляется содержимое шаблона charclass-select-layout.html.

Чтобы программное обеспечение конструктора могло правильно отображать ход выбора класса, необходимо создать на странице элементы со следующими id:

  • charclssel-input — hidden-поле, в которое будет записываться id выбранного класса;
  • charclssel-name — название текущего выбранного класса;
  • charclssel-description — описание текущего выбранного класса;
  • charclssel-error — сообщение об ошибке;
