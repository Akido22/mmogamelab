За оформление списка платных услуг отвечает файл paid-services.html в шаблоне оформления игры и аналогичный файл paid-services.html в шаблоне оформления социального интерфейса. Он может использовать следующие переменные шаблонизатора:

  • money — если присутствует, то необходимо отобразить количество денег, имеющееся на счетах персонажа. Это список, у каждого из элементов которого есть параметры:
    • html — HTML-представление остатка по одной из валют;
  • services — список доступных платных услуг. У каждой услуги есть параметры:
    • offers — список предложений по данной услуге. Каждый элемент списка имеет параметры:
      • html — HTML-представление предложения (строка вида "10 USD за 30 дней");
    • status — статус платной услуги (подключена/отключена);
    • status_cls — CSS-класс, которым необходимо оформить статус (может отсутствовать);
    • auto_prolong — если присутствует, то показать статус автопродления. Имеет поля:
      • status — текущий статус;
      • href — ссылка на действие по переключению статуса;
      • cmd — текст ссылки;
    • description — описание платной услуги.
