За оформление личного кабинета игрока отвечает файл cabinet.html в шаблоне оформления игры. Он может использовать следующие переменные шаблонизатора:

  • title — заголовок в кабинете;
  • create — необходимость отображать кнопку создания нового персонажа;
  • characters — если нужно показать список персонажей, то передаётся этот список. У каждого персонажа есть поля:
    • uuid — идентификатор персонажа;
    • name — его имя.
