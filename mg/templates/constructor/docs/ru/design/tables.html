В играх часто встречаются типовые интерфейсы, где необходимо просто отобразить таблицу без всяких украшений. Это может быть история продаж в магазине, список отправленных писем, история захватов замка или что-то аналогичное.

За оформление типовых таблиц отвечает файл tables.html в шаблоне оформления игры. Он может использовать следующие переменные шаблонизатора:

  • ret — если присутствует, то должна отобразиться ссылка возврата с адресом ret.href и текстом ret.html;
  • pages — если присутствует, то должен отобразиться блок выбора страниц. Это список, у каждого элемента которого есть параметры:
    • href — ссылка (может отсутствовать);
    • html — текст;
  • tables — список таблиц. Каждая таблица — это структура со следующими полями:
    • header — если присутствует, то необходимо отобразить заголовок таблицы. Поля каждой колонки заголовка:
      • html — текст;
      • cls — CSS-класс (может отсутствовать);
    • rows — список, каждым элементом которого является ещё один список, соответствующий колонкам одной строки. У каждой колонки есть поля:
      • html — текст;
      • cls — CSS-класс (может отсутствовать).
