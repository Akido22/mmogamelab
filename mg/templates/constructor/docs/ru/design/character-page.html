Внутриигровая страница персонажа является внутренней страницей игрового интерфейса. Шаблон задаётся в файле character-page.html игрового оформления. В нём можно использовать переменные шаблонизатора:

  • character.name — имя персонажа;
  • character.html — HTML-представление имени персонажа;
  • character.sex — пол персонажа (0 — мужской, 1 — женский);
  • character.avatar — HTML-представление аватара персонажа;
  • character.params — список всех параметров персонажа (может отсутствовать). Каждый параметр — это запись. Если у неё установлено поле header, то это заголовок. header содержит текст заголовка. Иначе — это информация о параметре, и запись имеет поля name (название) и value (значение);
  • character.actions — список ссылок. Каждый элемент списка — это запись с полями:
    • href — ссылка;
    • text — текст ссылки.

Аватар персонажа, который помещается на страницу информации, может содержать образ персонажа и, например, слоты экипировки вокруг него. Шаблон задаётся в файле character-page-avatar.html игрового оформления. В нём можно использовать переменные шаблонизатора:

  • avatar_image — ссылка на картинку с образом персонажа;
