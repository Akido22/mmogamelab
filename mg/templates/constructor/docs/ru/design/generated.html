Генератор оформлений

Генераторы оформлений предназначены для простого создания пользователями новых вариантов оформления для страниц своих игр. Для администратора игры генерация нового варианта оформления заключается в:

  • выборе шаблона из списка доступных;
  • загрузка крупного изображения по теме игры (если в шаблоне предусмотрено для него место);
  • ввод текста с названием игры, который необходимо разместить на шаблоне (если в шаблоне этот текст предусмотрен);
  • настройка параметров оформления (для разных шаблонов они могут быть разными): цвета фона, текста, дополнительных элементов.

Получив указанные данные, генератор сформирует эксклюзивный вариант оформления специально под данного администратора. Далее администратор может либо установить сгенерированный вариант, либо скачать себе его на компьютер, чтобы вручную произвести любые желаемые доработки.
