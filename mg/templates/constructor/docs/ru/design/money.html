За оформление списка счетов персонажа отвечает файл money-accounts.html в шаблоне оформления игры. Он может использовать следующие переменные шаблонизатора:

  • accounts — список счетов. У каждого счёта есть следующие параметры:
    • currency.name_plural — название валюты во множественном числе;
    • currency.image — изображение валюты (большое);
    • currency.icon — иконка валюты;
    • currency.description — описание валюты;
    • balance — остаток на счету;
    • balance_currency — название валюты в склонении, соответствующем числу balance;
    • menu — меню доступных операций. Это список, у каждого элемента которого есть параметры:
      • href — ссылка;
      • onclick — действие при клике (может отсутствовать);
      • html — текст ссылки.

За оформление истории операций по счёту отвечает файл money-operations.html в шаблоне оформления игры. Он может использовать следующие переменные шаблонизатора:

  • ret — если присутствует, то должна отобразиться ссылка возврата с адресом ret.href и текстом ret.html;
  • pages — если присутствует, то должен отобразиться блок выбора страниц. Это список, у каждого элемента которого есть параметры:
    • href — ссылка (может отсутствовать);
    • html — текст;
  • Performed, Amount, Balance, Description — заголовки столбцов. Можно заменить на свои;
  • operations — список операций. Каждый элемент имеет параметры:
    • performed — дата и время операции;
    • amount — сумма операции;
    • cls — CSS-класс, которым необходимо выделить сумму операции (может отсутствовать);
    • balance — остаток на счету после операции;
    • description — описание операции.
