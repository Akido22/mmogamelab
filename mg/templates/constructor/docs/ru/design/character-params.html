Внутриигровая страница параметров персонажа является внутренней страницей игрового интерфейса. Шаблон задаётся в файле character-params.html игрового оформления. В нём можно использовать переменные шаблонизатора:

  • character.params — список всех параметров персонажа (может отсутствовать). Каждый параметр — это запись. Если у неё установлено поле header, то это заголовок. header содержит текст заголовка. Иначе — это информация о параметре, и запись имеет поля name (название) и value (значение).
