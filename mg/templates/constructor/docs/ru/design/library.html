За оформление страниц библиотеки отвечает файл library.html в шаблоне оформления социальной части игры. Он может использовать следующие переменные шаблонизатора:

  • library_blocks — список информационных блоков, которые необходимо отобразить на странице (может отсутствовать). Если присутствует, то каждый его элемент — это HTML-текст блока;
  • library_content — собственно текст библиотечной статьи;

Подробная информация о настройке библиотеки приведена на странице Библиотека игры.
