За оформление внешних страниц интерфейса отвечает файл external.html в шаблоне оформления игры. Он может использовать следующие переменные шаблонизатора:

  • title — заголовок окна;
  • head — позволяет движку конструктора добавлять в раздел head какие-либо дополнительные стили и скрипты;
  • design_root — ссылка на каталог, в котором хранятся файлы из шаблона оформления игры;
  • content — полезное содержимое страницы, которую необходимо отобразить.
