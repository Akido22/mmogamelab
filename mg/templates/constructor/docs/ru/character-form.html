У вас есть возможность составить анкету персонажа, которую каждый сможет заполнить. Количество доступных полей и их типы вы можете задать через админку (пункт "Персонажи / Анкета персонажа"):

Расмотрим настройки подробнее:

  • тип поля — выбор режима, в котором будет работать данное поле. При подключении определённых модулей конструктора у вас могут добавиться пункты в этом списке. Но обычно доступен только один — "Пользовательское поле";
  • код поля — понадобится вам, если вы захотите получить этот параметр через скриптовый движок или использовать при оформлении информации о персонаже;
  • короткое описание — это подсказка для вас;
  • тип поля — доступны варианты:
    • текстовое поле — обычное однострочное;
    • большое текстовое поле — многострочное поле ввода (textarea);
    • выпадающий список — вы сможете вручную задать варианты выбора;
  • порядок сортировки — поля в анкете будут сортироваться в порядке увеличения этого параметра;
  • описание для игроков — это текст, который увидят игроки, которые будут просматривать анкету персонажа;
  • текст рядом с полем ввода в анкете — это текст, который можно использовать в качестве приглашения ко вводу. Например, написать не просто "Ваш девиз", а "Назови свой девиз, отважный путник";
  • показывать при регистрации — если установить эту галочку, то система будет требовать ввода значения в это поле во время регистрации персонажа;
  • это поле является обязательным с уровня — в игре часто бывает полезно, чтобы игроки указали больше информации о себе. Требовать этого сразу при регистрации опрометчиво, поскольку кто-то может бросить заполнять длинную анкету и уйти. А если требовать заполнение анкеты постепенно — на одном уровне одно, на другом другое, то анкета со временем заполнится полностью.
