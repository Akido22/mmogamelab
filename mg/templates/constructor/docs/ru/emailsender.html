Система массовых рассылок по игрокам

Администратор игры имеет возможность осуществлять массовые рассылки писем по игрокам. Это может быть полезно для того, чтобы уведомить давно не играющих игроков о нововведениях в игре, либо для иных способов повышения интереса пользователей к игре.

Чтобы иметь возможность осуществлять рассылки, необходимо подключить модуль "Массовые рассылки e-mail". После этого в меню "E-mail / Массовые рассылки" появится редактор писем. Для того, чтобы создать письмо, необходимо заполнить следующую форму:

  • Тема сообщения — строка, в которой работает движок шаблонизатора;
  • тело сообщения — полноценный HTML-редактор, также поддерживающий доступ к шаблонизатору. Картинки, вставленные в письмо, будут доставлены непосредственно в письме;
  • скриптовое условие — это механизм, позволяющий отправить письмо не всем персонажам, а только определённым. Например:
    • только девушкам: 
      char.sex == 1
    • только магам: 
      char.p_class == "mage"
    • только нубам: 
      char.p_level <= 3

Движок шаблонизатора позволяет воспользоваться специальной разметкой, чтобы кастомизировать отправляемое письмо. Например, так:

Привет, [%char.name%]!
Я вижу, ты давно не [%char.sex ? "заходила" : "заходил"%] в игру. Позволь расскать тебе о наших новостях.

Тестовая отправка

После того, как письмо подготовлено, администратору рекомендуется проверить его на свой адрес, чтобы убедиться, что оно корректно форматируется в почтовом клиенте. Для этого необходимо зайти в "опции доставки" и выбрать пункт "Тестовая доставка (администратору)":

  • администратору рекомендуется завести себе почтовые ящики на всех популярных сервисах и протестировать доставку на каждый из них;
  • если письмо кастомизируется под разные классы/уровни персонажей, то администратору рекомендуется проверить каждый из вариантов, установив своему персонажу соответствующие параметры.

Отправка по базе игроков

После того, как письмо протестировано, администратор может отправить его по всей базе игроков. Для этого необходимо зайти в "опции доставки" и выбрать пункт "Массовая доставка (всем пользователям)". Письмо попадёт на премодерацию сотрудниками конструктора и будет проверено на соответствие пользовательскому соглашению.

В зависимости от результата модерации, письмо будет либо отправлено, либо возвращено администратору игры. В колонке "Отправлено" либо будет написана статистика по отправке, либо причина отказа от модератора.
